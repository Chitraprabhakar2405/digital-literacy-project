## Module Summaries
The Portfolio is divided into five subtasks. These are as follows:

### Task 1 - Digital Literacy Infographic 
I created a one page visual infographic using Canva, as it is a useful tool for presenting information clearly and efficiently. It covers what digital literacy is, safe internet practices and professional online presence. The infographic is designed for first-year students to quickly understand key digital concepts.

### Task 2 - Student Digital Portfolio 
Professional portfolios on Github, LinkedIn and Kaggle are essential for students. In this task I created profiles on these platforms. Github hosts code, LinkedIn builds professional networks and connections and Kaggle supports data science learning.


### Task 3 - Coding and Collaboration Platforms
I completed a beginner level challenge on HackerRank and created a 5 - question Digital Literacy Awareness quiz using Google Forms to share with my batchmates.

**Google Form Quiz Link** - https://docs.google.com/forms/d/e/1FAIpQLScR7yLY3G8BCqWrPKBd0krNs79u4uJvfZ9-7W9DXJSp1UQ6fw/viewform?usp=publish-editor

### Task 4 - Email Etiquette and Social Media Guide 
In this task I drafted two Emails: one requesting an assignment extension and one expressing interest in an internship. Also created a social media Do's and Don'ts checklist for college students.

### Task 5 – Cybercrime Awareness: UPI/Online Payment Fraud
In this task I researched and wrote a detailed case study on UPI/Online Payment Fraud in India, including how the scam works step-by-step. Created a prevention checklist with 8+ actionable tips specifically for college students.
